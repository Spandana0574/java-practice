# Java Practice

This repository contains my Java learning exercises and small programs as I work through Java fundamentals.

## Contents

- **LessonA**: A program that counts the number of even and odd integers in a fixed array.

## How to Run

1. Open the project in VS Code with the Java Extension Pack installed.
2. Open the .java file you want to run.
3. Click the "Run" button above the `main` method, or run it from the terminal.

- Java 17
